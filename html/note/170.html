CS:GO 战绩板网页挂件

卜卜口日常2018-1-29

最近一年浪费了很多时间在 CSGO 这个游戏、所以再浪费点时间做个网页用的 CSGO 战绩板挂件~

样例

宽度自适应，特殊宽度有特殊处理，画了和官方差别很大的简化版段位图标（很丑，爱用不用?

使用方法

<style> 
.csgo-stat-box{ 
    display:block; 
    max-width:500px; 
    height:220px; 
    border-radius:4px; 
} 
</style> 
<iframe class="csgo-stat-box" src="//v2.mouto.org/csgo-stat/41791759?level=11" frameborder="0"></iframe>

替换链接中的 41791759 成需要挂件的 STEAM v3 ID，level 按照白银到大地球的排序 0-18 随需求设定。
然后将代码插入到需要挂件的页面以 iframe 形式引入，即可挂载~

STEAM v3 ID 可以到 https://steamdb.info/calculator/ 之类的网站获取，取 v3ID 的连续数字部分

思路

通过 API 抓取 CSGO 游戏数据
将数据存储到 mysql 中做永固存储，依照最后更新时间更新数据
将数据通过 Templet.js 格式化成 HTML 输出
